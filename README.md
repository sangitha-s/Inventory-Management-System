# Inventory-Management-System
A simple Inventory Management System developed in C++ using Object-Oriented Programming (OOP) concepts and STL vectors.

## Features
- Add new products with ID, Name, Quantity, and Price
- Display all products in the inventory
- Search for a product by ID
- Update product details and stock quantity
- Delete products from the inventory
- Store data dynamically using STL vectors
- Demonstrate OOP concepts such as Classes, Objects, and Encapsulation

## Tech Stack
- Language: C++
- Concepts: OOP, STL Vectors, Encapsulation
- Tools: Git, GitHub, Visual Studio Code

## How to Run
1. Compile the program:
   bash
   g++ main.cpp -o inventory
